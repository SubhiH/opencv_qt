### What we have done :

- [x] Gray Scale
- [x] Binary - White/Black
- [x] Histogram
- [x] Soble Operator
- [x] Canny Edge Detector
- [x] HSV Color Space
- [x] Normalize Block Filter
- [x] Gaussian Filter
- [x] Median Filter
